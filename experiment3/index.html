Description

Two procedural generators were built with p5.js: an over-world biome map and a dungeon room-and-corridor map. Each lives on its own page and can be opened with the buttons above.

Technical

Both generators share the same 8 √ó 8-pixel tileset. The over-world uses Perlin noise layers to assign biomes and animated water; the dungeon algorithm places non-overlapping
rooms and connects them with tunnels. Putting each sketch on its own page avoids global callback collisions in p5.

Reflection

I spent a considerable amount of time just trying to get the two scenes to render on the same page, but opted for having them render on seperate html scripts.
Splitting the demos simplified debugging and guaranteed each sketch runs without having to juggle the DOM. I enjoyed this project and had to spend more time
on this one, especially with grasping the concept of noise.
