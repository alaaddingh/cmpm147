Experiment 1 - Getting Started

Description

The goal of this experiment is to create a simple text-based generator for a silly potion-based story.

Technical

This was created using Javascript arrays in which random values are selected then concatenated into a small paragraph.

Reflection

I found this assignment to be a fun exercise as I mainly focused on trying to create humerous word-generations. I also enjoyed customizing the CSS to make it more visually appealing.
